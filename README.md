# todo-app_react
## Basic ToDo app on ReactJS

You can: 

* Add a task with title
* Edit a task
* Mark a task as completed
* Remove a task
* Persist tasks between working sessions with [React Simple Storage](https://www.npmjs.com/package/react-simple-storage)
