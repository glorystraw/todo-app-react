## Basic ToDo app on ReactJS

You can: 

- [x] Add a task with title
- [x] Edit a task
- [x] Mark a task as completed
- [x] Remove a task
- [x] Persist tasks between working sessions with LocalStorage
- [x] List tasks in sorted in reverse order by task title
